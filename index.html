Ejercicios intermedios

1. Diseño de Página de Inicio Responsiva: Crea una página de inicio que utilice el sistema de rejilla de Bootstrap para organizar un encabezado, una sección de características destacadas y un área de suscripción. Asegúrate de que la página se vea bien en dispositivos de diferentes tamaños.

Pagina de Inicio

Caracteristicas destacadas:

  1. Primera caracteristica destacada
  2. Segunda caracteristica destacada
  3. Tercera caracteristica destacada
  4. Cuarta caracteristica destacada

Suscribete

2. Menú de Navegación con Desplazamiento Suave: Desarrolla un menú de navegación que utilice la rejilla de Bootstrap y que incluya desplazamiento suave a secciones específicas de la página. Asegúrate de que el menú sea completamente responsivo para adaptarse a dispositivos móviles.

Nombre del sitio web
  • Inicio
  • Barra de navegacion
  • Formulario
  • Galería
  • Panel de administración

3. Galería de Imágenes con Lightbox Responsivo: Crea una galería de imágenes utilizando el sistema de rejilla de Bootstrap para organizar miniaturas. Al hacer clic en una miniatura, debería abrirse un lightbox que muestre la imagen en tamaño completo. Asegúrate de que la galería sea responsiva.

4. Formulario de Registro con Validación: Diseña un formulario de registro que utilice el sistema de rejilla de Bootstrap para organizar campos como nombre, correo electrónico y contraseña. Implementa validación en el lado del cliente para mejorar la experiencia del usuario.

5. Panel de Administración con Gráficos Interactivos: Construye un panel de administración utilizando la rejilla de Bootstrap. Integra gráficos interactivos (puedes utilizar bibliotecas como Chart.js) para visualizar datos relevantes. Asegúrate de que el panel sea responsivo y se vea bien en diferentes dispositivos.
